Image Utils
===========

Utility/Convenient API that handles the common **image** processing
such as read, write, image conversion, resize, etc.

The idea behind this utility module is to mimic the basic `Nuke`_
feature via python.

This library uses `OpenImageIO`_ at its core and utilizes
its API features
